# QuranBot Privacy Policy

Last Updated: 2026

This Privacy Policy explains how QuranBot handles user data.

## 1. Data Collected

QuranBot may collect and store the following limited information:

* Discord User IDs
* Discord Server IDs
* Voice channel IDs
* Bot configuration settings

This data is used only to operate the bot's features.

## 2. No Personal Data Collection

QuranBot does not collect personal information such as:

* Real names
* Email addresses
* Passwords
* Messages content

## 3. Data Usage

Stored data is used solely for:

* Maintaining bot functionality
* Remembering server settings
* Providing continuous playback features

## 4. Data Sharing

QuranBot does **not sell, trade, or share user data** with third parties.

## 5. Data Removal

Server administrators may request removal of stored server data by contacting the developer.

## 6. Security

Reasonable measures are taken to protect stored data from unauthorized access.

## 7. Changes

This policy may be updated when necessary. Continued use of the bot indicates acceptance of the updated policy.
